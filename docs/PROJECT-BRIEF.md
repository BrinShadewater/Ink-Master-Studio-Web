# Project Brief

## Purpose

Ink Master Studio Web is a browser-based production assistant for apparel and screen-print workflows. It is meant to reduce the friction between source artwork, mockup review, and exportable proof material.

## Audience

- Print-shop operators preparing proofs
- Designers checking artwork across garment colors
- Clients reviewing apparel mockups
- Maintainers improving upload, preview, export, and AI-assisted workflows

## Product Principles

- Production speed matters.
- Controls should map to real workflow decisions.
- Output should be easy to share.
- Security and cost controls matter because the app touches uploads and AI services.
- The UI should feel like a workbench, not a novelty demo.

## Core Workflows

- Upload artwork
- Choose or adjust mockup presets
- Preview artwork across garment colors
- Batch process variants
- Export proof material
- Review previous exports

## High-Risk Areas

- Client-side API-key exposure
- Malicious or oversized uploaded files
- SVG/script handling
- Export accuracy
- Mockup asset naming and alignment
- Unexpected AI API costs
